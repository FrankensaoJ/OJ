### AC答案

1.  原答案是在一开始用了

    ```cpp
    set<string> course[2504];
    ```
    
    AC答案换成了
    
    ```cpp
    vector<string> course[2504];
    //...
    
    sort(course[id].begin(), course[id].end());
    ```
    
    也就是输入管输入，排序管排序。可以看出，每次输入用set进行排序是很费时间的，会有多次无用查找。
    
    插入耗时: `log(n) vs 1`, n次插入方法n倍。
    
    因此，不如先全部顺序存储，然后一次性排序。
